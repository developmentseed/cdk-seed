# cdk-seed :seedling:

## Development

### Commands

#### `npx lerna bootstrap`

Install dependencies for each module.

#### `npx lerna run build`

Build each module.

#### `npx lerna run package`

Package each module for distribution.

#### `npx lerna run docgen`

Generate documentation for each module.
