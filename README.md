# cdk-seed :seedling:

## Development

### Commands

#### `npm run bootstrap`

Install dependencies for each package.

#### `npm run build`

Build each package.

#### `npm run package`

Prepare each package for distribution.

#### `npm run docgen`

Generate documentation for each package.

#### `npm test`

Run tests for all packages.
